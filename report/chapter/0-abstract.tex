\begin{abstract}
    Fight the Landlord is a challenging imperfect-information card game with a large combinatorial action space,
    hidden hands, and strong role-dependent cooperation and competition. In this project, we study several reinforcement
    learning and search-based approaches for building agents for this game.

    We first set the baseline:random and greedy agents. The random agent samples uniformly from the legal actions, while the greedy agent uses a small set of hand-crafted rules based on card combinations.

    We then explore reinforcement learning methods, including policy gradient and q-learning, to learn policies that can adapt to different game states and roles.

    Finally, we try adversarial search techniques, such as Monte Carlo Tree Search (MCTS), to improve decision-making by simulating future game states and evaluating potential actions.
