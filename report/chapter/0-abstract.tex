\begin{abstract}
Fight the Landlord is a challenging imperfect-information card game with a large combinatorial action space, hidden hands, and strong role-dependent cooperation and competition. In this project, we study several reinforcement learning and search-based approaches for building agents for this game. We first investigate value iteration and tabular value representations, but find that direct state enumeration is ineffective because the observable state must include not only the player's hand, but also public history, remaining card counts, turn context, and uncertainty over hidden cards. We then explore approximate Q-learning with history-aware features, neural policy learning through self-play, and adversarial search based on sampled determinizations of hidden hands. Finally, we combine learned Q-value estimators with adversarial search by using a trained Q model as the leaf evaluator. Our experiments and implementation process show that pure offline planning is difficult in this domain, while feature-based learning and bounded search provide more practical approximations for decision making under hidden information.
\end{abstract}
